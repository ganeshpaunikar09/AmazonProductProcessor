# Amazon Product Processor

This is a Java project that reads product data from a CSV file and processes it using Java Stream API.

Features:
- Read product data from CSV
- Find top rated product
- Find most expensive product per brand
- Filter products based on rating and price
- Calculate average price per brand
- Find most reviewed product
- Apply discount on product price

Technologies Used:
- Java
- Java Stream API
- File Handling
- Collections
